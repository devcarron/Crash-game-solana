# Solana Chess Casino Game

This is chess game on the Solana blockchain. As a play-to-earn game, I have implemented smart contract for security. Solana program was built by Anchor framework and UI is built by react.js. This is not full code, I have shared only UI and smart contract here. For full working website, feel free to reach out of me when you need supports [Twitter: https://x.com/blacksky_jose, Telegram: https://t.me/blacksky_jose, Discord: https://discord.gg/blacksky_jose].



## How to use it

```bash
git clone https://github.com/devcarron/Crash-game-solana.git
```

```bash
cd ./Solana-Chess-Casino-Game
```

```bash
npm run install
```

- For smart contract deployment:
```bash
anchor build
```

```bash
anchor deploy
```

- To start UI
```bash
cd ./frontend
```

```bash
npm run install
```

```bash
npm run start
```
